# EasyCarousel

A lightweight, customisable project showcase carousel built with plain HTML, CSS, and JavaScript.

## Overview

EasyCarousel is a finite card carousel designed to showcase projects in a clean, interactive way. Three cards are visible at a time, with the currently active card snapping to the center and appearing slightly larger than the others. Each card displays project information and a link to the project page.

## Features

- **Scroll snapping** – cards snap cleanly into position when scrolling
- **Scroll buttons** – previous/next buttons to navigate between cards
- **Anchor positioning** – precise control over card and button placement
- **Active card scaling** – the currently snapped card is visibly larger than the surrounding ones
- **Project info display** – the active card shows project details and a link to the project page
- **3 visible cards at a time** – consistent layout with one card always centered

## Tech Stack

- HTML
- CSS (no webkit, no vendor prefixes)
- JavaScript (vanilla, no frameworks)

No build tools required — it's a fully static project.

1. Clone the repository
2. Open `index.html` in your browser
